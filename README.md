This web application is a laboratory exercise for the course <i>Web Programming</i>. <br>
This is a simple <b>Spring Boot</b> E-Shop application, using Spring Security for in-memory authentication, 
Spring Data JPA for simple JPA based repositories, which access the underlying PostgreSQL database, and Thymeleaf is used as a template engine. 
Users with role admin have the privilege to add, edit and remove items (balloons), while regular users can only view the list of balloons and give specifications on the balloon they want to buy.
